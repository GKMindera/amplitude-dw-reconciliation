# amplitude-dw-reconciliation
Scripts that standardise how we compare Amplitude data vs DW data.

These scripts are what underpin the reconciliation dashboard found in [DataBricks](https://fdg-analytics.cloud.databricks.com/dashboardsv3/01f0ae0599b61f5fb1efa4f158937d1f/published).

Please use these SQL scripts (or the parameters thereof) to compare Amplitude and DW data.
